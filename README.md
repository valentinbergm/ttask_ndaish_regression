# Аналіз LTV користувачів (Cohort Analysis)

**Завдання:**
- Розрахунок Lifetime Value (LTV) за 52 тижні
- Прогноз доходу від когорт 39–43 тижня
- Оцінка точності моделі (R² та MAPE)

**Результати:**
- LTV за 1 рік: **1.3819 USD**
- Прогнозований дохід від когорт 39–43: **91,574 USD**
- Точність моделі: R² = 0.9703, MAPE (без Week 0) = 3.70%
